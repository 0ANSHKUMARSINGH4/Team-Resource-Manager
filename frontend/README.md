# Team Resource Manager

A full-stack application to manage developers, tasks, assignments, and workload.

## Tech Stack
- Backend: Spring Boot, JPA, MySQL
- Frontend: React + Chart.js
- Database: MySQL

## Features
- Developer management
- Task creation & assignment
- Workload calculation dashboard
- Capacity utilization charts

## How to Run
### Backend
```bash
cd backend
mvn spring-boot:run
```
### Frontend
```bash
cd frontend
npm install
npm start
```
